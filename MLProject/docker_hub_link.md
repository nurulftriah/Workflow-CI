# Tautan Docker Hub

Image Docker dari proyek ini tersedia secara publik di Docker Hub:

🔗 https://hub.docker.com/r/nunuft/eksperimen-sml-nurul

## Cara pull image

```bash
docker pull nunuft/eksperimen-sml-nurul:latest
```

## Cara run container

```bash
docker run nunuft/eksperimen-sml-nurul:latest
```

## Informasi Image

| Key        | Value                                      |
| ---------- | ------------------------------------------ |
| Repository | nunuft/eksperimen-sml-nurul           |
| Tag        | latest                                     |
| Base Image | python:3.12-slim                           |
| Deskripsi  | ML training image - Credit Scoring Project |
